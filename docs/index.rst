
Startrack
=========

The next generation of star trail imaging with Python.


Installation
------------

.. code-block:: bash

    python setup.py install

Testing
-------

Run the test:

.. code-block:: bash

    py.test startrack

Usage
-----

Import the module:

.. code-block:: python

    import startrack
    print(startrack.__version__)
